Google Apps Script — Preventivi su Drive

URL dello script che compila il template preventivo su Google Drive.

Google Apps Script — Fatture su Drive

URL dello script che compila il template fattura su Google Drive. Riceve tutti i dati della fattura: regime fiscale, IVA, scadenza, metodo di pagamento e note legali.

Se non configurato, viene usato lo script preventivi come fallback.
Come configurare gli script (una volta sola)
Script preventivi — HSLab_PreventivoScript.gs
  1. Apri script.google.com → Nuovo progetto
  2. Incolla il codice da HSLab_PreventivoScript.gs
  3. Distribuisci → Nuova distribuzione → App web
    Esegui come: Tu · Accesso: Tutti
  4. Copia l'URL /exec nel campo preventivi qui sopra
Script fatture — HSLab_FatturaScript.gs
  1. Crea un secondo progetto su script.google.com
  2. Incolla il codice da HSLab_FatturaScript.gs
  3. Distribuisci allo stesso modo → copia l'URL /exec
  4. Incollalo nel campo fatture qui sopra
Lo script riceve un oggetto JSON con i campi: tipo, numero, data, scadenza, regime, cliente, voci[], imponibile, iva, iva_perc, totale, metodo_pagamento, data_pagamento, note
